# ContextAPI Clean Arcitecture

node version 14.4.0

```
npm run start
```

```
yarn start
```

위의 명령어를 통해 localhost:3000 으로 로컬 환경에서 실행 가능합니다.

- Stacks
  - React를 통해 SPA 구현
  - React Context API를 통해 State management
  - Styled Component로 Style sheet 작성
  - TypeScript 로 정적 타입 분석
